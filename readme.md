# Anime Explorer

A simple web app to search and explore anime using the Jikan API.  
This project was built as part of a frontend milestone to practice JavaScript concepts like API handling, DOM manipulation, and higher-order functions.

---

# Features

- Search anime by name
- Filter anime by genre (Action, Comedy, Adventure)
- Sort anime by rating (ascending / descending)
- Like / Unlike anime
- Toggle Dark / Light mode
- Loading indicator while fetching data
- Direct link to view anime details

---

# Tech Stack

- HTML
- CSS
- JavaScript
- Jikan API (MyAnimeList unofficial API)

---

# Concepts Used

- DOM Manipulation
- Fetch API (async/await)
- Array Higher Order Functions:
  - map()
  - filter()
  - sort()
- Event Handling
- Responsive Design
