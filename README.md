Anime Explorer Web App

About the Project

Anime Explorer is a web application that allows users to search and explore anime using real-time data from a public API. Users can view anime details such as title, rating, episodes, genres, and description through a simple and interactive interface.

---

Purpose

The goal of this project is to learn how to work with APIs, asynchronous JavaScript, and build a dynamic web application that supports searching, filtering, and sorting data.

---
API Used

Jikan API (MyAnimeList API)
(https://api.jikan.moe/v4/anime?q=naruto)

The API provides anime information including:

* Title and images
* Ratings
* Genres
* Episodes
* Synopsis

Example endpoint:

```
https://api.jikan.moe/v4/anime?q=naruto
```

---

Planned Features

* Search anime by name
* Display anime results dynamically
* View anime details
* Filter anime by genre or type
* Sort anime by rating or popularity

---

Technologies

* HTML
* CSS
* JavaScript (ES6)
* Fetch API
* Git & GitHub


A simple anime search web app built using JavaScript and Jikan API to practice API integration, search, filtering, and sorting features.
